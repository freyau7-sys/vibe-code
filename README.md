# 🧠 脑力训练

一款专为中老年人设计的认知训练网页应用，帮助锻炼记忆力、反应力、语言力、专注力和逻辑力。

**在线体验：** `https://你的用户名.github.io/仓库名/`

---

## 功能介绍

共 5 个训练玩法，每局固定 5 轮，满分 100 分，结束后显示星级评价和历史最高分。

| 玩法 | 训练能力 | 规则 |
|------|---------|------|
| 🔢 数字记忆 | 记忆力 | 记住 5～9 位数字序列，消失后按顺序作答，每轮答对 +20 分 |
| ⚡ 反应速度 | 反应力 | 方块变绿后立刻点击，5 次平均反应时间换算为百分制得分 |
| 💬 词语联想 | 语言力 | 从 9 个词中找出属于指定类别的 3 个，全对 +20 分 |
| 🔍 专注搜寻 | 专注力 | 在 25 格图案中找出所有指定符号，错点一次扣 4 分 |
| 🧩 图形逻辑 | 逻辑力 | 观察序列规律，从 4 个选项中选出正确答案，每题 +20 分 |

## 评分标准

- ⭐⭐⭐ 80 分及以上
- ⭐⭐ 60～79 分
- ⭐ 60 分以下

历史最高分保存在本地浏览器，刷新页面不会丢失。

## 部署方法

### 方法一：GitHub Pages（推荐）

1. 新建 GitHub 仓库
2. 将 `index.html` 上传到仓库根目录
3. 进入仓库 **Settings → Pages → Source**，选择 `main` 分支，点击 Save
4. 等待约 1 分钟，访问生成的网址即可

### 方法二：本地使用

直接用浏览器打开 `index.html` 文件，无需服务器，无需联网。

---

## 添加到手机桌面

用 Safari（iPhone）或 Chrome（Android）打开网页后：

- **iPhone：** 点击底部分享按钮 → 「添加到主屏幕」
- **Android：** 点击菜单 → 「添加到主屏幕」

添加后图标出现在桌面，使用体验接近原生 App。

## 技术说明

- 纯原生 HTML / CSS / JavaScript，单文件，零依赖
- 使用系统自带字体（PingFang SC / Helvetica Neue），无需联网加载字体
- 数据存储于浏览器 `localStorage`，完全本地，不上传任何信息
- 兼容 iOS Safari、Android Chrome、桌面浏览器
